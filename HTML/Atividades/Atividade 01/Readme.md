Objetivo
Aplicar os conceitos de estrutura fundamental do HTML5, tags de texto, elementos de bloco/inline e a organização semântica de uma página.

Instruções Iniciais
Abra o editor de texto.
Crie uma nova pasta para Atividades
Crie um novo arquivo e salve-o como index.html.
Certifique-se de salvar o arquivo com a extensão .html para que o navegador consiga interpretá-lo.
Passo a Passo da Atividade

Etapa 1: A Estrutura Global
Comece estruturando o esqueleto do documento. Defina o idioma para português do Brasil e configure o título da aba do navegador.

Etapa 2: O Cabeçalho Semântico
Dentro da tag <body>, utilize a tag <header> para criar um cabeçalho.

Adicione um título principal (<h1>) com o nome: "Blog de Tecnologia do [Seu Nome]".
Crie uma barra de navegação simples usando <nav> e três links (<a>) fictícios: "Início", "Sobre" e "Contato".
Etapa 3: Conteúdo Principal e Formatação
Use a tag <main> para o conteúdo central. Dentro dela, crie um <article> sobre "O Surgimento da Web".

Use um <h2> para o título do artigo.
Escreva um parágrafo (<p>) explicando que o HTML foi criado por Tim Berners-Lee (use <strong> no nome dele).
Adicione um subtítulo <h3> chamado "Curiosidades".
Crie uma frase citando o CERN, usando a tag <i> para destacar o termo estrangeiro.
Utilize a tag <mark> para destacar o ano de 1989.
Etapa 4: Elementos de Bloco e Linha
Abaixo do artigo, adicione uma linha horizontal (<hr>).

Crie uma seção de "Avisos" usando <section>.
Escreva um parágrafo informando que um curso está com preço promocional: "De R$ 100,00 por R$ 50,00" (Dica: use as tags de riscado e negrito aprendidas).
Adicione a fórmula da água (H₂O) e um cálculo matemático simples (x²) usando as tags <sub> e <sup>.
Etapa 5: O Rodapé
Finalize a página com a tag <footer>.

Insira um parágrafo com o símbolo de copyright: &copy; 2026 - Desenvolvido por [Seu Nome].
Checklist de Verificação
Após terminar, abra o arquivo no seu navegador (Chrome, Edge ou Firefox) e verifique se:

O título na aba do navegador está correto.
O título principal (h1) é o maior da página.
O texto em negrito e itálico está aparecendo conforme o esperado.
A linha horizontal separa o conteúdo do artigo da seção de avisos.
Os links no cabeçalho aparecem lado a lado (comportamento inline).
Desafio Extra (Opcional)
Tente adicionar uma imagem logo após o primeiro parágrafo utilizando a tag <img src="..." alt="...">. Você pode usar o link de uma imagem da internet para testar!
